# The UDPipe Scripts

The UDPipe scripts help streaming motionCapture Data via udp proxy scripts (aka server) to other locations without the need to worry about firewalls and other nastyness.

---

## Content

NatNetPipeLocal: Sending Motive data to the udp proxy scripts (aka server)
NatNetPipeRemote: Receiving Motive data from the udp proxy scripts (aka server) and provide a connection point for clients (like unity)
